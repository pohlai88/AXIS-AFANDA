# AXIS-AFENDA — Landing Page Design

## Overall Design Direction

**Mood**

* Calm, quiet, spacious
* “Luxury control room”, not startup hype
* Low visual noise, high intentionality

**Layout principles**

* Large margins
* Fewer sections, more breathing room
* Slow scroll rhythm
* Every section answers *one emotional question*

**Color usage**

* Neutrals dominate (background, cards)
* Gold only appears:

  * once per viewport max
  * on direction / decision moments
* No gradients screaming for attention

---

## 1. HERO — *Orientation, not persuasion*

### Goal

Make the visitor **exhale** and feel understood in 3 seconds.

### Layout

```
[ left-aligned, not centered ]

AXIS-AFENDA
Headline
Subline
Primary CTA (soft)
Secondary CTA (text)
```

### Content

**Eyebrow (small, muted)**

```
THE COMPASS FOR WORK
```

**Hero Headline (large, calm, confident)**

```
Life is chaos.
Work doesn’t have to be.
```

**Subheadline (short, human, grounding)**

```
AXIS-AFENDA gives teams direction, structure,
and peace — without invading life.
```

**Primary CTA (quiet)**

```
[ See how work feels without noise ]
```

**Secondary CTA (text-only)**

```
Read the philosophy →
```

### Visual

* Background: `bg-lux-paper`
* One subtle compass / axis line motif (not literal icon)
* No screenshots yet

## 2. THE PAIN — *Recognition before solution*

### Goal

Make them think: *“They understand my daily fatigue.”*

### Layout

Three vertical blocks, not cards. Minimal borders.

### Content

**Section Title**

```
Modern work is exhausting — quietly.
```

**Block 1**

```
Too many tools.
No center.
```

> A hundred apps. Endless context switching.
> You spend more time finding work than doing it.

**Block 2**

```
Work never leaves you alone.
```

> Messages blur into life.
> Notifications demand attention at all hours.

**Block 3**

```
Nothing truly belongs to you.
```

> Decisions locked in platforms you can’t leave.
> Your work, owned by someone else.

No CTAs here.
Let it sit.
